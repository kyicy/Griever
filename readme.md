## Griever

Griever is a lightweight client for NetEase Music. 

Build upon:
- Electron
- Sequelize
- Sqlite3

### Current Version: 0.0.1
- [x] Search Music
- [x] Cache music and cover
- [x] Play Music after Search
- [x] Lyrics


### Next Version: 0.0.2
- [ ] Playlist
- [ ] Manage Cached files
- [ ] better UI
